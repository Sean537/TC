# ANSI 转义码参考手册

ANSI转义码用于控制终端文本样式、颜色和光标位置。本文档提供完整的参考和实用示例。

## 文本样式

| 转义码       | 功能描述                 | 示例                          | 兼容性说明                     |
|--------------|--------------------------|-------------------------------|--------------------------------|
| `\033[0m`    | 重置所有样式             | `echo -e "正常\033[1m加粗\033[0m"` | 所有主流终端支持               |
| `\033[1m`    | 加粗文本                 | `\033[1m加粗文本\033[0m`      | 广泛支持                       |
| `\033[3m`    | 斜体文本                 | `\033[3m斜体\033[0m`          | 部分终端不支持（如Windows CMD）|
| `\033[4m`    | 下划线文本               | `\033[4m下划线\033[0m`        | 广泛支持                       |
| `\033[7m`    | 反色显示                 | `\033[7m反色\033[0m`          | 部分终端支持                   |
| `\033[9m`    | 删除线文本               | `\033[9m删除线\033[0m`        | 部分终端支持                   |

## 颜色控制

### 基础8色
| 前景码       | 背景码       | 颜色   | 示例                          |
|--------------|--------------|--------|-------------------------------|
| `\033[30m`   | `\033[40m`   | 黑色   | `\033[30m黑字\033[0m`         |
| `\033[31m`   | `\033[41m`   | 红色   | `\033[31;41m红字红底\033[0m`  |

### 256色扩展
```bash
# 前景色示例 (橙色调)
echo -e "\033[38;5;202m橙色文字\033[0m"

# 背景色示例
echo -e "\033[48;5;123m天蓝背景\033[0m"
```

### RGB真彩色
```bash
# 自定义前景色 (RGB)
echo -e "\033[38;2;255;100;0m橙红色\033[0m"

# 自定义背景色
echo -e "\033[48;2;50;200;100m薄荷绿背景\033[0m"
```

## 光标控制

| 转义码          | 功能描述               | 示例                |
|-----------------|------------------------|---------------------|
| `\033[<n>A`     | 光标上移n行           | `\033[2A` 上移2行   |
| `\033[<n>B`     | 光标下移n行           | `\033[1B` 下移1行   |
| `\033[<行>;<列>H` | 移动光标到指定位置    | `\033[10;5H` 第10行第5列 |

## 特殊效果

| 转义码       | 功能描述               | 兼容性说明         |
|--------------|------------------------|--------------------|
| `\033[8m`    | 隐藏文本（安全输入用） | 部分终端支持       |
| `\033[?25l`  | 隐藏光标               | 需终端支持         |
| `\033[?25h`  | 显示光标               | 需终端支持         |

## 使用注意事项

1. **转义序列**：在bash中使用`-e`参数解释转义符，如`echo -e`
2. **兼容性测试**：重要功能应在目标终端测试
3. **重置样式**：使用`\033[0m`结束样式避免影响后续输出
4. **组合使用**：可组合多个代码，如`\033[1;31;43m`表示加粗红字黄底

```bash
# 综合示例：彩色进度条
for i in {1..10}; do
    echo -ne "\033[42m \033[0m"
    sleep 0.1
done
echo
```

> 提示：在支持真彩色的终端（如iTerm2、VS Code终端）可获得最佳视觉效果
