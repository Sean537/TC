| ANSI 转义符                 | 功能描述                                  | 示例/参数说明                            | 终端兼容性说明         |
|-----------------------------|-------------------------------------------|------------------------------------------|------------------------|
| `\033[0m`                   | 重置所有样式（颜色、背景等）              | `echo -e "\033[31m红字\033[0m"`          | 所有主流终端支持       |
| `\033[1m`                   | 加粗文本                                  | `\033[1mBold\033[0m`                     | 广泛支持               |
| `\033[3m`                   | 斜体文本                                  | `\033[3mItalic\033[0m`                   | 部分终端不支持（如Windows CMD） |
| `\033[4m`                   | 下划线文本                                | `\033[4mUnderline\033[0m`                | 广泛支持               |
| `\033[30m`–`\033[37m`       | 8色前景色（黑、红、绿、黄、蓝、紫、青、灰） | `\033[31mRed Text\033[0m`                | 所有终端支持           |
| `\033[40m`–`\033[47m`       | 8色背景色                                 | `\033[41mRed Background\033[0m`          | 所有终端支持           |
| `\033[38;5;<n>m`            | 256色前景色                               | `\033[38;5;202mOrange Text\033[0m`       | 需终端支持256色（如iTerm、VS Code） |
| `\033[48;5;<n>m`            | 256色背景色                               | `\033[48;5;202mOrange Bg\033[0m`         | 同上                   |
| `\033[38;2;<r>;<g>;<b>m`    | RGB前景色                                 | `\033[38;2;255;0;0mRed\033[0m`           | 需终端支持TrueColor（如现代终端） |
| `\033[48;2;<r>;<g>;<b>m`    | RGB背景色                                 | `\033[48;2;0;255;0mGreen Bg\033[0m`      | 同上                   |
| `\033[<n>A`                 | 光标上移`n`行（默认1）                    | `\033[2A` 上移2行                        | 广泛支持               |
| `\033[<n>B`                 | 光标下移`n`行（默认1）                    | `\033[1B` 下移1行                        | 广泛支持               |
| `\033[<n>C`                 | 光标右移`n`列（默认1）                    | `\033[5C` 右移5列                        | 广泛支持               |
| `\033[<n>D`                 | 光标左移`n`列（默认1）                    | `\033[3D` 左移3列                        | 广泛支持               |
| `\033[2J`                   | 清屏（保留滚动历史）                      | `echo -e "\033[2J"`                      | 广泛支持               |
| `\033[<行>;<列>H`           | 移动光标到指定行列                        | `\033[10;20H` 移动到第10行第20列         | 广泛支持               |
| `\033[?25l`                 | 隐藏光标                                  | `\033[?25l`                              | 需终端支持             |
| `\033[?25h`                 | 显示光标                                  | `\033[?25h`                              | 需终端支持             |
| `\033[7m`                   | 反色显示（前景与背景互换）                | `\033[7mInverse\033[0m`                  | 部分终端支持           |
| `\033[8m`                   | 隐藏文本（不可见但可复制）                | `\033[8mSecret\033[0m`                   | 部分终端支持           |
| `\033[9m`                   | 删除线文本                                | `\033[9mStrikethrough\033[0m`            | 部分终端支持           |
